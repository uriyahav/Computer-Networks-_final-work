Instructions how to run the code:
Open the terminal in the folder where the source files are located.
In the terminal use the command 'make'/'make all'- this  command will compile the code files to an object files called , prefix_table , prefix_table_opt.
Run this file with the txt file  you want to.
for example:
./prefix input.txt
./prefix_opt input.txt
for output example look at the running examples pdf file
note: there is already sample input file.
